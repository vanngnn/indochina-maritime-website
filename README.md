# Indochina Shipping Service and Forwarding Co. Ltd Website

Static marketing website for **Indochina Shipping Service and Forwarding Co. Ltd** with a navy/white theme and service sections for:

- Dunnage supply
- Visa and immigration support
- Crew change coordination
- Ship agency support

## Files Included

- `index.html` - Main single-page website content
- `styles/main.css` - Complete styling and responsive layout
- `scripts/main.js` - Mobile menu interaction and dynamic footer year
- `assets/logo.svg` - Brand logo graphic

## How to Run

Open `index.html` directly in your browser, or run a static server from this folder.
